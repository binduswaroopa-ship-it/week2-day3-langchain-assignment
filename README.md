# LangChain Assignment

## Overview

This project demonstrates the basic usage of the LangChain framework to build a simple AI-powered application. The application accepts user input, sends it to a Large Language Model (LLM), and returns the generated response.

## Objectives

* Understand the fundamentals of LangChain.
* Learn how to integrate Large Language Models.
* Build simple chains and prompts.
* Execute user queries through LangChain.

## Technologies Used

* Python 3.14+
* LangChain
* OpenAI API (or any supported LLM)
* Python Virtual Environment


## Prerequisites

* Python 3.14 or above
* pip package manager
* OpenAI API Key (optional depending on the model used)

## Installation

### 1. Clone the repository

```bash
git clone <repository-url>
cd langchain-assignment
```

### 2. Create a virtual environment

```bash
python -m venv venv
```

### 3. Activate the virtual environment

**Windows**

```bash
venv\Scripts\activate
```

**macOS/Linux**

```bash
source venv/bin/activate
```

## Environment Variables

Create a `.env` file and add your API key:

```env
OPENAI_API_KEY=your_api_key_here
```

## Running the Application

```bash
python app.py
```

